# Mapo CLI

Command-line tool for generating mapping code from C# source files outside the normal build pipeline.

## When to Use

Most users should use the **NuGet source generator** (`dotnet add package Mapo`), which runs automatically during `dotnet build`. The CLI is useful when you need to:

- Generate mappers in a separate build step or script
- Inspect generated code without enabling `EmitCompilerGeneratedFiles`
- Integrate Mapo into a non-MSBuild toolchain
- Pre-generate mappers for environments where source generators aren't available

## Installation

```bash
# Install globally
dotnet tool install --global Mapo.Cli

# Or install locally in your repo
dotnet tool install Mapo.Cli
```

## Usage

```
mapo gen <input-dir> <output-dir>
```

| Argument | Description |
|:---------|:------------|
| `<input-dir>` | Directory containing C# source files (scanned recursively) |
| `<output-dir>` | Directory where generated `.g.cs` files are written |

### Examples

```bash
# Generate mappers from src/ into a Generated/ directory
mapo gen src/ Generated/

# Generate from current directory
mapo gen . ./Generated

# Use absolute paths
mapo gen /projects/myapp/Models /projects/myapp/Generated
```

### Other Commands

```bash
mapo --help       # Show usage information
mapo --version    # Print version number
```

## How It Works

1. Scans `<input-dir>` recursively for all `.cs` files
2. Parses them into a Roslyn compilation
3. Finds all classes annotated with `[Mapper]`
4. Runs the same parser and emitter used by the source generator
5. Writes each generated mapper to `<output-dir>/{ClassName}.g.cs`

The generated code is identical to what the NuGet source generator produces during `dotnet build`.

## Example

Given this mapper in `src/UserMapper.cs`:

```csharp
using Mapo.Attributes;

[Mapper]
public partial class UserMapper
{
    public partial UserDto Map(User user);

    static void Configure(IMapConfig<User, UserDto> config)
    {
        config.Map(d => d.FullName, s => s.FirstName + " " + s.LastName);
    }
}
```

Run:

```bash
mapo gen src/ generated/
```

Output in `generated/UserMapper.g.cs`:

```csharp
// <auto-generated/>
public partial class UserMapper
{
    public partial UserDto Map(User user)
    {
        if (user == null) throw new ArgumentNullException(nameof(user));
        return MapInternal(user);
    }

    [MethodImpl(MethodImplOptions.AggressiveInlining | MethodImplOptions.AggressiveOptimization)]
    private UserDto MapInternal(User user)
    {
        var target = new UserDto();
        target.FullName = user.FirstName + " " + user.LastName;
        // ...
        return target;
    }
}
```
